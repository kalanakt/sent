# Media and Files

Send and download photos, documents, videos, and other media with **sent**.

## Send a file

```python
await client.send_file("username", "photo.jpg", caption="Check this out")
```

Force as document (no compression):

```python
await client.send_file("username", "report.pdf", force_document=True)
```

Send file with a message in one call:

```python
await client.send_message("username", "Here's the file", file="data.zip")
```

See [`send_file()`](/api/client/files#send_file) and [`send_message()`](/api/client/messages#send_message).

## Upload only

Upload to Telegram CDN without sending:

```python
input_file = await client.upload_file("large.zip")
```

See [`upload_file()`](/api/client/files#upload_file).

## Download media

```python
path = await client.download_media(message)
path = await client.download_media(message, "downloads/photo.jpg")
```

Download profile photo:

```python
path = await client.download_profile_photo("username")
```

See [`download_media()`](/api/client/files#download_media).

## Progress callback

Pass `progress_callback` to `send_file` or `download_media` for upload/download progress:

```python
def progress(current, total):
    print(f"{current}/{total}")

await client.send_file(chat, "big.mp4", progress_callback=progress)
```
