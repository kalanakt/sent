## Handling flood waits

```python
from sent.errors import FloodWaitError

try:
    await client.send_message(chat, text)
except FloodWaitError as e:
    await asyncio.sleep(e.seconds)
```

Set `flood_sleep_threshold` on the client to auto-sleep for short waits.
